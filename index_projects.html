PROYECTOS

RRSS DE PORSCHE ALICANTE

En este proyecto se crearon distintas publicaciones para las redes sociales del Centro Porsche de Alicante.

ALTITUD 38

Este proyecto muestra el packaging para una gama gourmet que reinterpreta la tradición alicantina desde una mirada contemporánea. Este proyecto esta conjuntamente realizado con el diseñador Ángel Tovar Sánchez.

POSTERS DEL MOTORSPORT

En este proyecto se presentan los mejores posters creados para presentar distintas carreras entre competiciones como la Formula 1, el WRC o el GTWC de Europa.
ALSSENCE

Este proyecto es la creación de una marca para una cafetería en Alicante que combina café de calidad con un espacio de creatividad, autoexpresión y comunidad.

MERCADO CENTRAL ALICANTE

El siguiente proyecto muestra el diseño de una nueva identidad visual para el Mercado Central de Alicante, reconocido por su calidad y compromiso con la comunidad.

RRSS DE MAYA WEUG

En este proyecto se ilustraron tres posters para las redes sociales de Maya Weug, piloto de ferrari, para anunciar distintas carreras de la temporada 2024.

TRILOGÍA CARMEN MOLA

El siguiente proyecto trata sobre el diseño de tres portadas para la misma trilogía policiaca, tres libros terriblemente escalofriantes.

ÀNIMA D'UN REBEL

El siguiente proyecto muestra el diseño de la etiqueta para un vino con alma de rebelde para Bodegas Riko.

CIRCUIT RICARDO TORMO

Este proyecto muestra fotografias tomadas en distintos eventos realizados en el Circuit Ricardo Tormo. Eventos dinámicos y de gran emoción, todo representado a través de la cámara.

EL PLACER ARTÍSTICO

El siguiente proyecto muestra distintos posters de distintos temas: desde los problemas que más afectan a los españoles a diseños creados por amor al arte.
